## 1. P2: 三层 Agent 架构（核心加分项）

### 借鉴来源
**Toonflow** 的 Workflow Engine 设计

### 实现文件
- `server/agents/scriptAgent.js` - 三层 Agent 核心实现
- `server/skills/script_execution_skeleton.md` - 故事骨架生成 Skill
- `server/skills/script_execution_script.md` - 剧本写作 Skill
- `server/skills/script_supervision_review.md` - 质量评估 Skill

### 架构设计
```
┌─────────────────────────────────────────────────────────────┐
│                    Decision Layer (决策层)                   │
│  • 接收用户输入，orchestrate 整个流程                          │
│  • 协调 Execution 和 Supervision 层                           │
│  • 根据评分结果决定是否重试                                    │
└──────────────────────┬──────────────────────────────────────┘
                       │
        ┌──────────────┴──────────────┐
        ▼                              ▼
┌──────────────────────┐    ┌──────────────────────┐
│  Execution Layer     │    │  Supervision Layer   │
│  (执行层)             │    │  (监督层)             │
│                      │    │                      │
│  • 生成故事骨架        │    │  • 质量评估 (A/B/C)   │
│  • 编写分镜剧本        │    │  • 问题检测           │
│  • 修订剧本           │    │  • 改进建议           │
└──────────────────────┘    └──────────────────────┘
```

### 代码示例
```javascript
class DecisionAgent {
  async run() {
    // 1. 执行层生成剧本
    const executionAgent = new ExecutionAgent(productInfo, materials);
    const skeleton = await executionAgent.generateSkeleton();
    const script = await executionAgent.generateScript(skeleton);
    
    // 2. 监督层评估
    const supervisionAgent = new SupervisionAgent();
    const review = await supervisionAgent.review(script);
    
    // 3. 根据评分决定流程
    if (review.score >= 'B') {
      return script;  // 通过
    } else {
      return executionAgent.reviseScript(script, review.suggestions);  // 重试
    }
  }
}
```

## 2. P1-2: 分镜编辑器数据结构（轨道/时间轴）

### 借鉴来源
**Toonflow** 的 Track-based Timeline 设计

### 实现文件
- `server/index.js` - `/api/storyboard/tracks` 路由
- `my-app/src/App.tsx` - 前端轨道可视化

### 核心逻辑
每个轨道最长 **15秒**，超过自动分组到新轨道：

```javascript
// 轨道计算算法
for (const scene of scenes) {
  const duration = scene.duration || 3;
  
  if (currentDuration + duration > 15 && currentTrack.length > 0) {
    tracks.push({ id: tracks.length + 1, scenes: currentTrack, totalDuration: currentDuration });
    currentTrack = [];
    currentDuration = 0;
  }
  
  currentTrack.push({
    ...scene,
    trackStartTime: currentDuration,
    trackEndTime: currentDuration + duration
  });
  currentDuration += duration;
}
```

### 前端展示
- Timeline 组件展示分镜顺序
- 轨道卡片展示分组情况
- Tooltip 显示配音内容

## 3. P1-3: 多分镜视频拼接（FFmpeg）

### 借鉴来源
**Toonflow** 的 VideoComposer 设计

### 实现文件
- `server/services/videoComposer.js` - VideoComposer 类

### 核心功能
1. **视频拼接** - 多个分镜视频合并
2. **转场效果** - fade/dissolve/cut
3. **音频混合** - TTS + BGM
4. **轨道分组** - 15秒/轨道

### FFmpeg 命令生成
```javascript
buildTransitionFilter(sceneVideos, transition) {
  let filter = '';
  let lastV = '[0:v]';
  let lastA = '[0:a]';
  
  for (let i = 1; i < sceneVideos.length; i++) {
    // 视频转场
    if (transition === 'fade') {
      filter += `${lastV}[${i}:v]xfade=transition=fade:duration=0.5:offset=${duration}[v${i}];`;
    }
    // 音频淡入淡出
    filter += `${lastA}[${i}:a]acrossfade=d=0.5[a${i}];`;
    
    lastV = `[v${i}]`;
    lastA = `[a${i}]`;
  }
  
  return filter;
}
```

### API 端点
```
POST /api/video/compose
{
  "scenes": [...],
  "options": {
    "transition": "fade",  // fade/dissolve/cut
    "bgm": "path/to/bgm.mp3"
  }
}
```

```javascript
const result = await ttsService.generate('欢迎使用AIGC系统', {

  outputDir: './temp'

// 返回结果
{
  audioFile: './temp/audio_xxx.mp3',
  subtitleFile: './temp/audio_xxx.srt',
  duration: 3.5
}
```

```
   │              │               │              │            ▼
   │              │               │              │      ┌──────────┐
   │              │               │              │      │ 最终视频  │
   │              │               │              ▼      └──────────┘
   │              │               │         ┌──────────┐
   │              │               │         │ FFmpeg   │
   │              │               │         │ 拼接视频  │
   │              │               ▼         └──────────┘

   │              │         │ Seedance │
   │              │         │ 视频生成  │
   │              ▼         └──────────┘
   │        ┌──────────┐
```

### 批量生成流程
```javascript
// 1. 为每个分镜生成视频
for (let i = 0; i < scenes.length; i++) {

// 2. 生成 TTS 配音
```
