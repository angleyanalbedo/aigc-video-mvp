## 一、项目背景与目标

### 1.1 现有系统问题

| 问题 | 说明 |
|------|------|
| 步骤引导式流程 | 用户必须按素材→剧本→创作→预览四步走，无法自由跳转 |
| Agent 能力分散 | ScriptAgent、VideoAgent、ClipAgent 独立运行，缺乏统一调度 |
| 黑盒操作 | 用户不知道 Agent 干了什么，无法追踪操作历史 |
| 缺乏交互性 | Agent 只能执行固定流程，无法响应用户自由指令 |

### 1.2 创新目标

构建一个 **AI Native 的智能无限画布工作台**，实现：

1. **自由对话式交互** - 用户用自然语言下达指令，Agent 自动理解并执行
2. **可视化思维过程** - Agent 的意图识别、任务规划、操作执行全程可视化
3. **双向实时同步** - 画布状态与数据库实时同步，用户操作与 Agent 操作互不影响
4. **异步确认机制** - 危险的写操作需要用户确认，保障数据安全
5. **无限画布布局** - 不受线性流程限制，支持节点自由拖拽和组织

## 三、数据库设计

### 3.1 新增数据表

#### 3.1.1 画布节点表（canvas_nodes）

```sql
CREATE TABLE canvas_nodes (
  id TEXT PRIMARY KEY,
  project_id TEXT NOT NULL,
  node_type TEXT NOT NULL,  -- 'script' | 'scene' | 'material' | 'video' | 'intent' | 'plan' | 'operation' | 'comment'
  node_data TEXT NOT NULL,  -- JSON 格式的节点数据
  position_x REAL NOT NULL,
  position_y REAL NOT NULL,
  width REAL DEFAULT 200,
  height REAL DEFAULT 150,
  style TEXT,               -- CSS 样式（颜色、边框等）
  metadata TEXT,            -- 额外元数据
  created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
  updated_at DATETIME DEFAULT CURRENT_TIMESTAMP,
  FOREIGN KEY (project_id) REFERENCES projects(id) ON DELETE CASCADE
);

CREATE INDEX idx_canvas_nodes_project ON canvas_nodes(project_id);
CREATE INDEX idx_canvas_nodes_type ON canvas_nodes(node_type);
```

#### 3.1.2 节点连接表（canvas_connections）

```sql
CREATE TABLE canvas_connections (
  id TEXT PRIMARY KEY,
  project_id TEXT NOT NULL,
  source_node_id TEXT NOT NULL,
  target_node_id TEXT NOT NULL,
  connection_type TEXT NOT NULL,  -- 'timeline' | 'reference' | 'dependency' | 'operation' | 'alternative'
  label TEXT,                      -- 连接线标签（可选）
  style TEXT,                      -- 线条样式
  metadata TEXT,
  created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
  FOREIGN KEY (project_id) REFERENCES projects(id) ON DELETE CASCADE,
  FOREIGN KEY (source_node_id) REFERENCES canvas_nodes(id) ON DELETE CASCADE,
  FOREIGN KEY (target_node_id) REFERENCES canvas_nodes(id) ON DELETE CASCADE
);

CREATE INDEX idx_connections_project ON canvas_connections(project_id);
CREATE INDEX idx_connections_source ON canvas_connections(source_node_id);
CREATE INDEX idx_connections_target ON canvas_connections(target_node_id);
```

#### 3.1.3 操作历史表（operation_history）

```sql
CREATE TABLE operation_history (
  id TEXT PRIMARY KEY,
  project_id TEXT NOT NULL,
  session_id TEXT,                       -- 对话会话 ID
  agent_name TEXT NOT NULL,              -- 'MasterAgent' | 'ScriptAgent' | 'VideoAgent' | etc.
  operation_type TEXT NOT NULL,          -- 'create' | 'update' | 'delete' | 'generate' | 'compose'
  target_node_id TEXT,                   -- 操作的目标节点 ID
  source_intent_id TEXT,                 -- 关联的意图节点 ID
  operation_details TEXT NOT NULL,       -- JSON: 完整的操作详情
  status TEXT DEFAULT 'pending',         -- 'pending_confirmation' | 'confirmed' | 'executing' | 'completed' | 'failed' | 'cancelled'
  user_confirmation BOOLEAN DEFAULT FALSE,
  result TEXT,                           -- 执行结果
  error TEXT,                            -- 错误信息
  created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
  confirmed_at DATETIME,
  executed_at DATETIME,
  completed_at DATETIME,
  FOREIGN KEY (project_id) REFERENCES projects(id) ON DELETE CASCADE
);

CREATE INDEX idx_operations_project ON operation_history(project_id);
CREATE INDEX idx_operations_status ON operation_history(status);
CREATE INDEX idx_operations_session ON operation_history(session_id);
CREATE INDEX idx_operations_agent ON operation_history(agent_name);
```

```javascript
// server/agents/masterAgent.js

const { EventEmitter } = require('events');
const IntentParser = require('./intent/intentParser');
const TaskPlanner = require('./planner/taskPlanner');
const ToolExecutor = require('./executor/toolExecutor');
const CanvasSyncService = require('../services/canvasSyncService');
const { llmProvider } = require('../services/providers');

class MasterAgent extends EventEmitter {
  constructor() {
    super();
    this.name = 'Copilot Master Agent';
    this.intentParser = new IntentParser();
    this.taskPlanner = new TaskPlanner();
    this.toolExecutor = new ToolExecutor();
    this.canvasSync = new CanvasSyncService();
    this.activeSessions = new Map();  // sessionId -> session context
  }

  /**
   * 处理用户消息
   * @param {string} message - 用户消息
   * @param {string} projectId - 项目 ID
   * @param {object} context - 上下文信息
   * @returns {Promise<AgentResponse>}
   */
  async processMessage(message, projectId, context = {}) {
    console.log(`\n🤖 MasterAgent: 收到消息 "${message.slice(0, 50)}..."`);
    
    // 1. 加载会话上下文
    const sessionContext = await this.loadSessionContext(projectId, context.sessionId);
    
    // 2. 意图识别
    const intent = await this.intentParser.parse(message, sessionContext);
    console.log(`🎯 意图识别: ${intent.primaryIntent}, 置信度: ${intent.confidence}`);
    
    // 3. 生成执行计划
    const plan = await this.taskPlanner.generatePlan(intent, projectId, sessionContext);
    console.log(`📋 生成计划: ${plan.steps.length} 个步骤`);
    
    // 4. 记录到画布
    const canvasUpdate = await this.canvasSync.createIntentAndPlan(
      projectId,
      intent,
      plan,
      context.sessionId
    );
    
    // 5. 发送实时更新给前端
    this.emit('canvasUpdate', canvasUpdate);
    
    // 6. 判断是否需要确认
    if (plan.requiresConfirmation && intent.intents.some(i => 
      ['generate_video', 'generate_image', 'generate_script', 'delete'].includes(i)
    )) {
      return {
        type: 'plan_confirmation',
        intentNodeId: canvasUpdate.intentNodeId,
        planNodeId: canvasUpdate.planNodeId,
        message: this.generateConfirmationMessage(plan),
        plan: plan,
        estimatedDuration: plan.estimatedDuration
      };
    }
    
    // 7. 直接执行（简单操作无需确认）
    return await this.executePlan(plan, projectId, canvasUpdate);
  }

  /**
   * 执行确认的计划
   */
  async executeConfirmedPlan(planNodeId, projectId) {
    console.log(`🚀 MasterAgent: 开始执行计划 ${planNodeId}`);
    
    // 1. 获取计划节点
    const plan = await this.canvasSync.getPlan(planNodeId);
    if (!plan) {
      throw new Error(`计划节点不存在: ${planNodeId}`);
    }
    
    // 2. 更新状态为执行中
    await this.canvasSync.updatePlanStatus(planNodeId, 'executing');
    
    // 3. 执行步骤
    const results = [];
    for (const step of plan.steps) {
      if (step.status === 'skipped') continue;
      
      // 创建操作节点
      const operationNodeId = await this.canvasSync.createOperationNode(
        projectId,
        step,
        planNodeId
      );
      
      try {
        // 更新步骤状态
        await this.canvasSync.updateStepStatus(planNodeId, step.stepId, 'executing');
        
        // 执行操作
        const result = await this.toolExecutor.execute(step, projectId, {
          onProgress: (progress) => {
            this.emit('operationProgress', {
              operationNodeId,
              progress,
              stepId: step.stepId
            });
          }
        });
        
        // 标记成功
        await this.canvasSync.updateStepStatus(planNodeId, step.stepId, 'completed', result);
        await this.canvasSync.updateOperationNode(operationNodeId, {
          status: 'completed',
          result,
          endTime: Date.now()
        });
        
        results.push({ stepId: step.stepId, success: true, result });
        
      } catch (error) {
        // 标记失败
        await this.canvasSync.updateStepStatus(planNodeId, step.stepId, 'failed', null, error.message);
        await this.canvasSync.updateOperationNode(operationNodeId, {
          status: 'failed',
          error: error.message,
          endTime: Date.now()
        });
        
        results.push({ stepId: step.stepId, success: false, error: error.message });
        
        // 如果是关键步骤失败，可能需要停止执行
        if (step.critical) {
          console.error(`❌ 关键步骤失败，停止执行`);
          break;
        }
      }
    }
    
    // 4. 更新计划状态
    const allSuccess = results.every(r => r.success);
    await this.canvasSync.updatePlanStatus(planNodeId, allSuccess ? 'completed' : 'failed');
    
    // 5. 生成执行报告
    return {
      planNodeId,
      success: allSuccess,
      results,
      summary: this.generateExecutionSummary(results)
    };
  }

  /**
   * 加载会话上下文
   */
  async loadSessionContext(projectId, sessionId) {
    const cacheKey = `${projectId}:${sessionId}`;
    
    if (this.activeSessions.has(cacheKey)) {
      return this.activeSessions.get(cacheKey);
    }
    
    // 从数据库加载历史
    const project = await projectModel.getById(projectId);
    const recentMessages = await this.canvasSync.getRecentMessages(projectId, sessionId, 10);
    const canvasNodes = await this.canvasSync.getNodes(projectId);
    
    const context = {
      project,
      recentMessages,
      canvasNodes,
      timestamp: Date.now()
    };
    
    this.activeSessions.set(cacheKey, context);
    return context;
  }

  generateConfirmationMessage(plan) {
    const stepDescriptions = plan.steps
      .map((s, i) => `${i + 1}. ${s.description}`)
      .join('\n');
    
    return `我将执行以下 ${plan.steps.length} 个操作：\n\n${stepDescriptions}\n\n预计耗时：${Math.ceil(plan.estimatedDuration / 60)} 分钟\n\n是否确认执行？`;
  }

  generateExecutionSummary(results) {
    const success = results.filter(r => r.success).length;
    const failed = results.filter(r => !r.success).length;
    
    if (failed === 0) {
      return `✅ 所有 ${success} 个操作执行成功！`;
    }
    
    return `⚠️ 执行完成：${success} 成功，${failed} 失败`;
  }
}

module.exports = new MasterAgent();
```

```javascript
class IntentParser {
  constructor() {

        ],
        weight: 0.9
      },

          /调整顺序/i,
          /重新排序/i,
          /调换.*顺序/i,
          /把.*放在.*前面/i
        ],
        weight: 0.9
      },

      'add_material': {
        patterns: [
          /上传.*素材/i,
          /添加.*图片/i,
          /引入.*材料/i,
          /上传.*商品图/i
        ],
        weight: 1.0
      },
      'search_material': {
        patterns: [
          /搜索.*素材/i,
          /找.*图片/i,
          /检索.*材料/i,
          /有没有.*素材/i
        ],
        weight: 0.9
      },
      
      // 剪辑类意图
      'compose_video': {
        patterns: [
          /剪辑/i,
          /合成.*视频/i,
          /拼接.*视频/i,
          /把.*合在一起/i
        ],
        weight: 1.0
      },
      'add_audio': {
        patterns: [
          /添加.*音乐/i,
          /配上.*配音/i,
          /加.*背景音/i,
          /添加.*音效/i
        ],
        weight: 1.0
      },
      
      // 信息查询
      'query_status': {

          /进展.*如何/i,
          /完成了.*吗/i,
          /生成.*到哪了/i
        ],

      },

        ],

      },
      
      // 删除类
      'delete': {

          /删除.*分镜/i,
          /移除.*素材/i,
          /删掉/i
        ],
        weight: 0.9,
        requiresConfirmation: true
      }
    };
    
    // 实体提取规则
    this.entityExtractors = {
      sceneId: /第(\d+)个?分镜/i,
      field: /(镜头类型|时长|旁白|描述|转场)/i,
      shotType: /(特写|中景|全景|俯拍|仰拍)/i,
      emotion: /(热情|专业|平静|幽默|震惊|温情)/i
    };
  }

  /**
   * 解析用户意图
   */
  async parse(message, context) {
    // 1. 意图分类
    const intents = this.detectIntents(message);
    
    // 2. 实体提取
    const entities = this.extractEntities(message, context);
    
    // 3. 置信度评估
    const confidence = this.calculateConfidence(intents, entities);
    
    // 4. 判断是否需要确认
    const requiresConfirmation = this.checkRequiresConfirmation(intents);
    
    // 5. 生成意图描述
    const intentDescription = this.generateIntentDescription(intents, entities);
    
    return {
      primaryIntent: intents[0]?.intent || 'unknown',
      intents: intents,
      entities,
      confidence,
      requiresConfirmation,
      intentDescription,
      originalMessage: message,
      timestamp: Date.now()
    };
  }

  /**
   * 检测意图
   */
  detectIntents(message) {
    const results = [];
    
    for (const [intentName, config] of Object.entries(this.intentPatterns)) {
      for (const pattern of config.patterns) {
        if (pattern.test(message)) {
          results.push({
            intent: intentName,
            matchedPattern: pattern.source,
            weight: config.weight,
            requiresConfirmation: config.requiresConfirmation || false
          });
          break;
        }
      }
    }
    
    // 按权重排序
    results.sort((a, b) => b.weight - a.weight);
    
    return results;
  }

  /**
   * 提取实体
   */
  extractEntities(message, context) {
    const entities = {};
    
    // 提取分镜 ID
    const sceneIdMatch = message.match(this.entityExtractors.sceneId);
    if (sceneIdMatch) {
      entities.sceneId = parseInt(sceneIdMatch[1]);
    }
    
    // 提取字段名
    const fieldMatch = message.match(this.entityExtractors.field);
    if (fieldMatch) {
      entities.field = fieldMatch[1];
    }
    
    // 提取镜头类型
    const shotMatch = message.match(this.entityExtractors.shotType);
    if (shotMatch) {
      entities.shotType = shotMatch[1];
    }
    
    // 提取情感
    const emotionMatch = message.match(this.entityExtractors.emotion);
    if (emotionMatch) {
      entities.emotion = emotionMatch[1];
    }
    
    // 从上下文推断目标
    if (context.canvasNodes) {
      // 如果提到了"这个"或"当前"，获取当前选中的节点
      const selectedNode = context.canvasNodes.find(n => n.selected);
      if (selectedNode) {
        entities.targetNodeId = selectedNode.id;
        entities.targetType = selectedNode.type;
      }
    }
    
    return entities;
  }

  calculateConfidence(intents, entities) {
    if (intents.length === 0) {
      return 0.3;  // 低置信度
    }
    
    let confidence = intents[0].weight;
    
    // 如果提取到关键实体，增加置信度
    if (entities.sceneId) confidence += 0.1;
    if (entities.field) confidence += 0.1;
    
    return Math.min(confidence, 1.0);
  }

  checkRequiresConfirmation(intents) {
    // 生成类和删除类操作需要确认
    const confirmationIntents = ['generate_video', 'generate_image', 'generate_script', 'delete', 'compose_video'];
    
    return intents.some(i => 
      confirmationIntents.includes(i.intent) || i.requiresConfirmation
    );
  }

  generateIntentDescription(intents, entities) {
    if (intents.length === 0) {
      return '未知意图';
    }
    
    const intentMap = {
      'generate_video': '生成视频',
      'generate_image': '生成图片',
      'generate_script': '生成剧本',
      'edit_scene': '编辑分镜',
      'edit_script': '编辑剧本',
      'reorder': '调整顺序',
      'add_material': '添加素材',
      'search_material': '搜索素材',
      'compose_video': '剪辑视频',
      'add_audio': '添加音频',
      'query_status': '查询状态',
      'explain': '解释说明',
      'delete': '删除'
    };
    
    let description = intentMap[intents[0].intent] || intents[0].intent;
    
    if (entities.sceneId) {
      description += `（分镜${entities.sceneId}）`;
    }
    
    if (entities.field) {
      description += `的${entities.field}`;
    }
    
    return description;
  }
}
```

```javascript
    plan.description = '生成视频';
    
    // 1. 如果没有剧本，先生成剧本
    if (!script || !script.scenes || script.scenes.length === 0) {
      plan.steps.push(this.createStep({
        type: 'generate_script',
        agent: 'ScriptAgent',
        description: '生成剧本和分镜',
        critical: true
      }));
      plan.estimatedDuration += 30;
    }
    
    // 2. 确定要生成视频的分镜
    let targetScenes = [];
    if (intent.entities.sceneId) {
      // 指定了分镜
      const scene = script?.scenes?.find(s => s.id === intent.entities.sceneId);
      if (scene) {
        targetScenes = [scene];
      }
    } else if (intent.entities.targetType === 'scene') {
      // 指定了分镜节点
      targetScenes = script?.scenes?.filter(s => !s.videoUrl) || [];
    } else {
      // 所有缺少视频的分镜
      targetScenes = script?.scenes?.filter(s => !s.videoUrl) || [];
    }
    
    // 3. 为每个目标分镜生成视频
    for (const scene of targetScenes) {
      plan.steps.push(this.createStep({
        type: 'generate_video',
        agent: 'VideoAgent',
        description: `生成分镜 ${scene.id} 的视频`,
        params: { sceneId: scene.id, scene },
        targetNodeId: `scene_${scene.id}`,
        critical: false
      }));
      plan.estimatedDuration += 30;  // 假设每个视频 30 秒
    }
    
    // 4. 剪辑合成（如果有多个分镜）
    if (targetScenes.length > 1) {
      plan.steps.push(this.createStep({
        type: 'compose_video',
        agent: 'ClipAgent',
        description: '剪辑合成最终视频',
        dependsOn: targetScenes.map(s => `generate_video_${s.id}`),
        critical: false
      }));
      plan.estimatedDuration += 60;
    }
  }

  /**
   * 规划：编辑分镜
   */
  async planEditScene(intent, project, plan) {
    const script = project.script;
    
    if (!intent.entities.sceneId) {
      throw new Error('未指定要编辑的分镜');
    }
    
    const scene = script?.scenes?.find(s => s.id === intent.entities.sceneId);
    if (!scene) {
      throw new Error(`分镜 ${intent.entities.sceneId} 不存在`);
    }
    
    plan.description = `编辑分镜 ${intent.entities.sceneId}`;
    plan.requiresConfirmation = false;  // 编辑操作无需确认
    
    // 确定要更新的字段
    const updates = {};
    
    if (intent.entities.field === 'shot_type' && intent.entities.shotType) {
      updates.shot_type = intent.entities.shotType;
    } else if (intent.entities.field === '时长' && intent.entities.value) {
      updates.duration = parseInt(intent.entities.value);
    }
    
    // 从消息中提取更多更新
    if (/改成|改为/.test(intent.originalMessage)) {
      if (/特写/.test(intent.originalMessage)) updates.shot_type = '特写';
      if (/中景/.test(intent.originalMessage)) updates.shot_type = '中景';
      if (/全景/.test(intent.originalMessage)) updates.shot_type = '全景';
      if (/\d+秒|\d+秒/.test(intent.originalMessage)) {
        const durationMatch = intent.originalMessage.match(/(\d+)秒/);
        if (durationMatch) updates.duration = parseInt(durationMatch[1]);
      }
    }
    
    plan.steps.push(this.createStep({
      type: 'update_node',
      agent: 'MasterAgent',
      description: `修改分镜 ${intent.entities.sceneId}`,
      params: {
        sceneId: intent.entities.sceneId,
        updates
      },
      targetNodeId: `scene_${intent.entities.sceneId}`
    }));
    
    plan.estimatedDuration = 1;  // 编辑操作很快
  }

  /**
   * 规划：生成剧本
   */
  async planGenerateScript(intent, project, plan) {
    plan.description = '生成剧本';
    plan.requiresConfirmation = true;
    
    plan.steps.push(this.createStep({
      type: 'generate_script',
      agent: 'ScriptAgent',
      description: '生成带货剧本和分镜',
      critical: true,
      params: {
        productInfo: project.product_info,
        existingScript: project.script
      }
    }));
    
    plan.estimatedDuration = 30;
  }

  /**
   * 规划：查询状态
   */
  async planQueryStatus(intent, project, plan) {
    plan.description = '查询项目状态';
    plan.requiresConfirmation = false;
    
    plan.steps.push(this.createStep({
      type: 'query_status',
      agent: 'MasterAgent',
      description: '查询当前生成状态',
      critical: false
    }));
    
    plan.estimatedDuration = 1;
  }

  createStep(config) {
    this.stepCounter++;
    return {
      stepId: `step_${Date.now()}_${this.stepCounter}`,
      type: config.type,
      agent: config.agent,
      description: config.description,
      targetNodeId: config.targetNodeId,
      params: config.params || {},
      dependsOn: config.dependsOn || [],
      status: 'pending',
      critical: config.critical || false,
      result: null,
      error: null
    };
  }
}

module.exports = TaskPlanner;
```

```javascript
// server/agents/executor/toolExecutor.js

const scriptAgent = require('../scriptAgent');
const videoAgent = require('../videoAgent');
const clipAgent = require('../clipAgent');
const imageAgent = require('../imageAgent');
const { updateSceneAsset } = require('../tools/workbenchAPI');
const projectModel = require('../../models/project');

class ToolExecutor {
  constructor() {

      ScriptAgent: scriptAgent,
      VideoAgent: videoAgent,

      ImageAgent: imageAgent
    };
  }

  /**
   * 执行单个步骤
   */
  async execute(step, projectId, callbacks = {}) {
    const { onProgress } = callbacks;
    
    console.log(`⚙️ 执行步骤: ${step.description}`);
    
    try {
      let result;
      
      switch (step.type) {
        case 'generate_script':
          result = await this.executeGenerateScript(step, projectId, onProgress);
          break;
          
        case 'generate_video':
          result = await this.executeGenerateVideo(step, projectId, onProgress);
          break;
          
        case 'generate_image':
          result = await this.executeGenerateImage(step, projectId, onProgress);
          break;
          
        case 'update_node':
          result = await this.executeUpdateNode(step, projectId, onProgress);
          break;
          
        case 'compose_video':
          result = await this.executeComposeVideo(step, projectId, onProgress);
          break;
          
        case 'query_status':
          result = await this.executeQueryStatus(projectId);
          break;
          
        default:
          throw new Error(`未知步骤类型: ${step.type}`);
      }
      
      return result;
      
    } catch (error) {
      console.error(`❌ 步骤执行失败: ${error.message}`);
      throw error;
    }
  }

  /**
   * 执行：生成剧本
   */
  async executeGenerateScript(step, projectId, onProgress) {
    onProgress?.({ message: '正在生成剧本...' });
    
    const productInfo = step.params.productInfo;
    const script = await scriptAgent.generate(productInfo, projectId);
    
    // 更新数据库
    await projectModel.update(projectId, { script });
    
    // 更新画布节点
    const scriptNodeId = step.targetNodeId || `script_${Date.now()}`;
    await canvasSyncService.upsertScriptNode(projectId, scriptNodeId, script);
    
    onProgress?.({ 
      message: '剧本生成完成', 
      data: { sceneCount: script.scenes?.length }
    });
    
    return { script, scriptNodeId };
  }

  /**
   * 执行：生成视频
   */
  async executeGenerateVideo(step, projectId, onProgress) {
    const { sceneId, scene } = step.params;
    
    onProgress?.({ message: `正在生成第 ${sceneId} 个分镜视频...` });
    
    // 调用 VideoAgent
    const result = await videoAgent.generateScene(scene, {
      projectId,
      sceneIndex: sceneId - 1,
      resolution: '720p',
      ratio: '9:16'
    }, (progress) => {
      onProgress?.({
        message: `生成分镜 ${sceneId} 视频 (${progress.current}/${progress.total})`,
        progress: progress.progress
      });
    });
    
    // VideoAgent 会自动调用 workbenchAPI 更新数据库
    // 这里只需要更新画布节点
    await canvasSyncService.updateSceneNode(projectId, `scene_${sceneId}`, {
      videoUrl: result.videoUrl,
      status: result.status === 'succeeded' ? 'completed' : 'failed'
    });
    
    onProgress?.({
      message: `分镜 ${sceneId} 视频生成${result.status === 'succeeded' ? '成功' : '失败'}`,
      data: result
    });
    
    return result;
  }

  /**
   * 执行：更新节点
   */
  async executeUpdateNode(step, projectId, onProgress) {
    const { sceneId, updates } = step.params;
    
    console.log(`📝 更新分镜 ${sceneId}:`, updates);
    
    // 调用 workbenchAPI 直接更新数据库
    for (const [field, value] of Object.entries(updates)) {
      await updateSceneAsset(projectId, sceneId - 1, field, value);
    }
    
    // 更新画布节点
    await canvasSyncService.updateSceneNode(projectId, `scene_${sceneId}`, updates);
    
    onProgress?.({ message: `分镜 ${sceneId} 更新完成` });
    
    return { success: true, updates };
  }

  /**
   * 执行：剪辑合成
   */
  async executeComposeVideo(step, projectId, onProgress) {
    onProgress?.({ message: '正在剪辑合成视频...' });
    
    const project = await projectModel.getById(projectId);
    const script = project.script;
    
    // 获取所有已生成的视频
    const videos = script.scenes
      .filter(s => s.videoUrl)
      .map(s => ({
        sceneId: s.id,
        videoUrl: s.videoUrl
      }));
    
    if (videos.length === 0) {
      throw new Error('没有可用的视频进行剪辑');
    }
    
    // 调用 ClipAgent
    const result = await clipAgent.execute(script, videos, {});
    
    // 创建视频节点
    const videoNodeId = `video_${Date.now()}`;
    await canvasSyncService.createVideoNode(projectId, videoNodeId, {
      videoUrl: result.video,
      duration: result.duration,
      status: 'completed',
      sourceScenes: videos.map(v => v.sceneId)
    });
    
    onProgress?.({
      message: '视频剪辑完成',
      data: { videoUrl: result.video, duration: result.duration }
    });
    
    return { videoUrl: result.video, duration: result.duration, videoNodeId };
  }

  /**
   * 执行：查询状态
   */
  async executeQueryStatus(projectId) {
    const project = await projectModel.getById(projectId);
    const script = project.script;
    
    if (!script || !script.scenes) {
      return {
        status: 'no_script',
        message: '暂无剧本'
      };
    }
    
    const stats = {
      total: script.scenes.length,
      completed: script.scenes.filter(s => s.status === 'completed').length,
      generating: script.scenes.filter(s => s.status === 'generating').length,
      pending: script.scenes.filter(s => s.status === 'pending').length,
      failed: script.scenes.filter(s => s.status === 'failed').length

    return {
      status: 'ok',
      stats,
      message: `总 ${stats.total} 个分镜，已完成 ${stats.completed} 个，正在生成 ${stats.generating} 个`
    };
  }
}
```

```javascript
// server/services/canvasSyncService.js

const db = require('../db');
const projectModel = require('../models/project');

class CanvasSyncService {
  constructor() {
    this.eventEmitter = null;  // WebSocket emitter
  }

  setEventEmitter(emitter) {
    this.eventEmitter = emitter;
  }

  // ─────────────────────────────────────────────────────────
  // 节点操作
  // ─────────────────────────────────────────────────────────

  /**
   * 创建节点
   */
  async createNode(projectId, nodeType, nodeData, position, style = {}) {
    const nodeId = `node_${Date.now()}_${Math.random().toString(36).slice(2, 8)}`;
    
    await db.run(`
      INSERT INTO canvas_nodes 
      (id, project_id, node_type, node_data, position_x, position_y, style)
      VALUES (?, ?, ?, ?, ?, ?, ?)
    `, [
      nodeId,
      projectId,
      nodeType,
      JSON.stringify(nodeData),
      position.x,
      position.y,
      JSON.stringify(style)
    ]);

    const node = {
      id: nodeId,
      projectId,
      type: nodeType,
      data: nodeData,
      position,
      style
    };

    // 广播更新
    this.broadcast(projectId, {
      type: 'node_created',
      node
    });

    return node;
  }

  /**
   * 更新节点
   */
  async updateNode(projectId, nodeId, updates) {
    const node = await this.getNode(nodeId);
    if (!node) {
      throw new Error(`节点不存在: ${nodeId}`);
    }

    const newData = { ...node.data, ...updates };
    
    await db.run(`
      UPDATE canvas_nodes 
      SET node_data = ?, updated_at = ?
      WHERE id = ?
    `, [JSON.stringify(newData), new Date().toISOString(), nodeId]);

    // 广播更新
    this.broadcast(projectId, {
      type: 'node_updated',
      nodeId,
      updates,
      newData
    });

    return { ...node, data: newData };
  }

  /**
   * 删除节点
   */
  async deleteNode(projectId, nodeId) {
    // 删除关联的连接
    await db.run(`
      DELETE FROM canvas_connections 
      WHERE source_node_id = ? OR target_node_id = ?
    `, [nodeId, nodeId]);

    // 删除节点
    await db.run('DELETE FROM canvas_nodes WHERE id = ?', [nodeId]);

    // 广播更新
    this.broadcast(projectId, {
      type: 'node_deleted',
      nodeId
    });

    return true;
  }

  /**
   * 获取节点
   */
  async getNode(nodeId) {
    const row = await db.get('SELECT * FROM canvas_nodes WHERE id = ?', [nodeId]);
    if (!row) return null;

    return this.parseNode(row);
  }

  /**
   * 获取项目的所有节点
   */
  async getNodes(projectId) {
    const rows = await db.all(
      'SELECT * FROM canvas_nodes WHERE project_id = ? ORDER BY created_at',
      [projectId]
    );

    return rows.map(row => this.parseNode(row));
  }

  parseNode(row) {
    return {
      id: row.id,
      projectId: row.project_id,
      type: row.node_type,
      data: JSON.parse(row.node_data),
      position: { x: row.position_x, y: row.position_y },
      size: { width: row.width, height: row.height },
      style: row.style ? JSON.parse(row.style) : {},
      metadata: row.metadata ? JSON.parse(row.metadata) : {},
      createdAt: row.created_at,
      updatedAt: row.updated_at
    };
  }

  // ─────────────────────────────────────────────────────────
  // 连接线操作
  // ─────────────────────────────────────────────────────────

  /**
   * 创建连接
   */
  async createConnection(projectId, sourceNodeId, targetNodeId, connectionType, options = {}) {
    const connectionId = `conn_${Date.now()}_${Math.random().toString(36).slice(2, 8)}`;

    await db.run(`
      INSERT INTO canvas_connections
      (id, project_id, source_node_id, target_node_id, connection_type, label, style)
      VALUES (?, ?, ?, ?, ?, ?, ?)
    `, [
      connectionId,
      projectId,
      sourceNodeId,
      targetNodeId,
      connectionType,
      options.label || null,
      JSON.stringify(options.style || {})
    ]);

    const connection = {
      id: connectionId,
      projectId,
      sourceNodeId,
      targetNodeId,
      connectionType,
      label: options.label,
      style: options.style
    };

    this.broadcast(projectId, {
      type: 'connection_created',
      connection
    });

    return connection;
  }

  /**
   * 获取项目的所有连接
   */
  async getConnections(projectId) {
    const rows = await db.all(
      'SELECT * FROM canvas_connections WHERE project_id = ?',
      [projectId]
    );

    return rows.map(row => ({
      id: row.id,
      projectId: row.project_id,
      sourceNodeId: row.source_node_id,
      targetNodeId: row.target_node_id,
      connectionType: row.connection_type,
      label: row.label,
      style: row.style ? JSON.parse(row.style) : {}
    }));
  }

  // ─────────────────────────────────────────────────────────
  // Agent 特定操作
  // ─────────────────────────────────────────────────────────

  /**
   * 创建意图和计划节点
   */
  async createIntentAndPlan(projectId, intent, plan, sessionId) {
    // 1. 创建 IntentNode
    const intentNode = await this.createNode(
      projectId,
      'intent',
      {
        intent: intent.primaryIntent,
        originalMessage: intent.originalMessage,
        entities: intent.entities,
        confidence: intent.confidence,
        sessionId
      },
      { x: 100, y: 50 },
      { borderColor: '#1890ff', backgroundColor: '#e6f7ff' }
    );

    // 2. 创建 PlanNode
    const planNode = await this.createNode(
      projectId,
      'plan',
      {
        steps: plan.steps,
        description: plan.description,
        estimatedDuration: plan.estimatedDuration,
        status: 'pending_confirmation',
        parentIntentId: intentNode.id
      },
      { x: 100, y: 250 },
      { borderColor: '#faad14', backgroundColor: '#fffbe6' }
    );

    // 3. 创建 Intent → Plan 连接
    await this.createConnection(
      projectId,
      intentNode.id,
      planNode.id,
      'operation',
      { label: '生成计划' }
    );

    // 4. 为每个步骤创建 OperationNode
    const operationNodes = [];
    let yOffset = 450;
    
    for (const step of plan.steps) {
      const operationNode = await this.createNode(
        projectId,
        'operation',
        {
          operationType: step.type,
          agentName: step.agent,
          description: step.description,
          targetNodeId: step.targetNodeId,
          status: 'pending',
          dependsOn: step.dependsOn
        },
        { x: 100, y: yOffset },
        { borderColor: '#52c41a', backgroundColor: '#f6ffed' }
      );

      // 创建 Plan → Operation 连接
      await this.createConnection(
        projectId,
        planNode.id,
        operationNode.id,
        'operation',
        { label: step.description }
      );

      operationNodes.push(operationNode);
      yOffset += 120;
    }

    return {
      intentNodeId: intentNode.id,
      planNodeId: planNode.id,
      operationNodeIds: operationNodes.map(n => n.id)
    };
  }

  /**
   * 更新计划状态
   */
  async updatePlanStatus(planNodeId, status) {
    const planNode = await this.getNode(planNodeId);
    if (!planNode) return;

    await this.updateNode(planNode.projectId, planNodeId, {
      status,
      ...(status === 'confirmed' ? { confirmedAt: Date.now() } : {}),
      ...(status === 'completed' ? { completedAt: Date.now() } : {})
    });
  }

  /**
   * 更新步骤状态
   */
  async updateStepStatus(planNodeId, stepId, status, result = null, error = null) {
    const planNode = await this.getNode(planNodeId);
    if (!planNode) return;

    const steps = [...planNode.data.steps];
    const stepIndex = steps.findIndex(s => s.stepId === stepId);
    
    if (stepIndex >= 0) {
      steps[stepIndex] = {
        ...steps[stepIndex],
        status,
        result: result || steps[stepIndex].result,
        error: error || steps[stepIndex].error,
        ...(status === 'executing' ? { startTime: Date.now() } : {}),
        ...(status === 'completed' || status === 'failed' ? { endTime: Date.now() } : {})
      };

      await this.updateNode(planNode.projectId, planNodeId, { steps });
    }
  }

  /**
   * 创建操作节点
   */
  async createOperationNode(projectId, step, parentPlanId) {
    const operationNode = await this.createNode(
      projectId,
      'operation',
      {
        operationType: step.type,
        agentName: step.agent,
        description: step.description,
        targetNodeId: step.targetNodeId,
        status: 'pending',
        params: step.params
      },
      { x: 100, y: Date.now() % 500 + 500 }
    );

    await this.createConnection(
      projectId,
      parentPlanId,
      operationNode.id,
      'operation'
    );

    return operationNode.id;
  }

  /**
   * 更新操作节点
   */
  async updateOperationNode(operationNodeId, updates) {
    const node = await this.getNode(operationNodeId);
    if (!node) return;

    await this.updateNode(node.projectId, operationNodeId, updates);
  }

  // ─────────────────────────────────────────────────────────
  // 业务表同步
  // ─────────────────────────────────────────────────────────

  /**
   * 同步剧本到画布
   */
  async syncScriptToCanvas(projectId, script) {
    // 创建或更新 ScriptNode
    const existingScriptNodes = (await this.getNodes(projectId))
      .filter(n => n.type === 'script');
    
    let scriptNodeId;
    if (existingScriptNodes.length > 0) {
      scriptNodeId = existingScriptNodes[0].id;
      await this.updateNode(projectId, scriptNodeId, script);
    } else {
      const scriptNode = await this.createNode(
        projectId,
        'script',
        script,
        { x: 50, y: 50 }
      );
      scriptNodeId = scriptNode.id;
    }

    // 为每个分镜创建 SceneNode
    const existingSceneNodes = (await this.getNodes(projectId))
      .filter(n => n.type === 'scene');
    
    for (let i = 0; i < script.scenes.length; i++) {
      const scene = script.scenes[i];
      const sceneNodeId = `scene_${scene.id}`;
      
      const existingNode = existingSceneNodes.find(n => n.id === sceneNodeId);
      if (existingNode) {
        await this.updateNode(projectId, sceneNodeId, scene);
      } else {
        const sceneNode = await this.createNode(
          projectId,
          'scene',
          scene,
          { x: 350, y: 50 + i * 180 }
        );
        
        // 创建 Script → Scene 连接
        await this.createConnection(
          projectId,
          scriptNodeId,
          sceneNode.id,
          'timeline'
        );
      }
    }

    return scriptNodeId;
  }

  // ─────────────────────────────────────────────────────────
  // 实时推送
  // ─────────────────────────────────────────────────────────

  broadcast(projectId, event) {
    if (this.eventEmitter) {
      this.eventEmitter.to(`canvas:${projectId}`).emit('canvasEvent', event);
    }
  }
}

module.exports = new CanvasSyncService();
```

```tsx
  planNodeId: string;
  plan: {
    steps: Array<{
      stepId: string;
      description: string;
      agent: string;
      status: string;
    }>;
    estimatedDuration: number;
  };
}

const CopilotChat: React.FC<{ projectId: string }> = ({ projectId }) => {
  const [messages, setMessages] = useState<Message[]>([]);
  const [inputValue, setInputValue] = useState('');
  const [isLoading, setIsLoading] = useState(false);
  const [pendingPlan, setPendingPlan] = useState<PlanConfirmation | null>(null);
  const messagesEndRef = useRef<HTMLDivElement>(null);
  const wsRef = useRef<WebSocket | null>(null);

  // WebSocket 连接
  useEffect(() => {
    wsRef.current = new WebSocket(`ws://${location.host}/ws/canvas/${projectId}`);
    
    wsRef.current.onmessage = (event) => {
      const data = JSON.parse(event.data);
      handleWebSocketMessage(data);
    };

    return () => {
      wsRef.current?.close();
    };
  }, [projectId]);

  // 自动滚动到底部
  useEffect(() => {
    messagesEndRef.current?.scrollIntoView({ behavior: 'smooth' });
  }, [messages]);

  const handleWebSocketMessage = (data: any) => {
    switch (data.type) {
      case 'operation_progress':
        setMessages(prev => [...prev, {
          id: `op_${Date.now()}`,
          role: 'system',
          type: 'operation_log',
          content: `⚙️ ${data.message}`,
          timestamp: Date.now()
        }]);
        break;
        
      case 'plan_completed':
        setMessages(prev => [...prev, {
          id: `complete_${Date.now()}`,
          role: 'assistant',
          type: 'text',
          content: `✅ 执行完成！${data.summary || ''}`,
          timestamp: Date.now()
        }]);
        setPendingPlan(null);
        break;
        
      case 'plan_failed':
        setMessages(prev => [...prev, {
          id: `failed_${Date.now()}`,
          role: 'assistant',
          type: 'error',
          content: `❌ 执行失败：${data.error}`,
          timestamp: Date.now()
        }]);
        setPendingPlan(null);
        break;
    }
  };

  const sendMessage = async () => {
    if (!inputValue.trim() || isLoading) return;

    const userMessage: Message = {
      id: `user_${Date.now()}`,
      role: 'user',
      type: 'text',
      content: inputValue,
      timestamp: Date.now()
    };

    setMessages(prev => [...prev, userMessage]);
    setInputValue('');
    setIsLoading(true);

    try {
      const response = await fetch('/api/agent/copilot', {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify({
          message: inputValue,
          projectId,
          sessionId: getSessionId()
        })
      });

      const result = await response.json();

      if (result.type === 'plan_confirmation') {
        // 显示计划确认卡片
        setPendingPlan({
          planNodeId: result.planNodeId,
          plan: result.plan
        });

        setMessages(prev => [...prev, {
          id: `assistant_${Date.now()}`,
          role: 'assistant',
          type: 'text',
          content: result.message,
          timestamp: Date.now()
        }]);
      } else {
        setMessages(prev => [...prev, {
          id: `assistant_${Date.now()}`,
          role: 'assistant',
          type: 'text',
          content: result.message,
          timestamp: Date.now(),
          metadata: result.data
        }]);
      }
    } catch (error) {
      setMessages(prev => [...prev, {
        id: `error_${Date.now()}`,
        role: 'assistant',
        type: 'error',
        content: '抱歉，发生了错误，请稍后重试。',
        timestamp: Date.now()
      }]);
    } finally {
      setIsLoading(false);
    }
  };

  const confirmPlan = async () => {
    if (!pendingPlan) return;

    setMessages(prev => [...prev, {
      id: `system_${Date.now()}`,
      role: 'system',
      type: 'operation_log',
      content: '🚀 开始执行计划...',
      timestamp: Date.now()
    }]);

    try {
      await fetch('/api/agent/copilot/execute', {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify({
          planNodeId: pendingPlan.planNodeId,
          projectId
        })
      });
    } catch (error) {
      console.error('执行计划失败:', error);
    }

    setPendingPlan(null);
  };

  const cancelPlan = async () => {
    if (!pendingPlan) return;

    try {
      await fetch('/api/agent/copilot/cancel', {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify({
          planNodeId: pendingPlan.planNodeId,
          projectId
        })
      });
    } catch (error) {
      console.error('取消计划失败:', error);
    }

    setPendingPlan(null);
    setMessages(prev => [...prev, {
      id: `system_${Date.now()}`,
      role: 'system',
      type: 'operation_log',
      content: '❌ 已取消执行',
      timestamp: Date.now()
    }]);
  };

  return (
    <div className="copilot-chat">
      {/* 消息列表 */}
      <div className="message-list">
        {messages.map(msg => (
          <MessageBubble key={msg.id} message={msg} />
        ))}
        
        {isLoading && !pendingPlan && (
          <div className="message assistant">
            <Spin indicator={<LoadingOutlined style={{ fontSize: 24 }} spin />} />
            <span>正在分析意图...</span>
          </div>
        )}
        
        <div ref={messagesEndRef} />
      </div>

      {/* 计划确认卡片 */}
      {pendingPlan && (
        <Card className="plan-confirmation-card" size="small">
          <div className="plan-header">
            <RobotOutlined /> 执行计划确认
          </div>
          <List
            size="small"
            dataSource={pendingPlan.plan.steps}
            renderItem={(step, index) => (
              <List.Item>
                <Tag color="blue">{index + 1}</Tag>
                <span>{step.description}</span>
                <Tag>{step.agent}</Tag>
              </List.Item>
            )}
          />
          <div className="plan-footer">
            <span>预计耗时: {Math.ceil(pendingPlan.plan.estimatedDuration / 60)} 分钟</span>
            <Button 
              type="primary" 
              icon={<CheckCircleOutlined />}
              onClick={confirmPlan}
            >
              确认执行
            </Button>
            <Button 
              danger 
              icon={<CloseCircleOutlined />}
              onClick={cancelPlan}
            >
              取消
            </Button>
          </div>
        </Card>
      )}

      {/* 输入框 */}
      <div className="input-area">
        <TextArea
          value={inputValue}
          onChange={e => setInputValue(e.target.value)}
          onPressEnter={e => {
            if (!e.shiftKey) {
              e.preventDefault();
              sendMessage();
            }
          }}
          placeholder="输入指令，如：帮我生成一个破壁机带货视频"
          autoSize={{ minRows: 1, maxRows: 4 }}
          disabled={isLoading}
        />
        <Button 
          type="primary" 
          icon={<SendOutlined />}
          onClick={sendMessage}
          disabled={!inputValue.trim() || isLoading}
        >
          发送
        </Button>
      </div>
    </div>
  );
};

      <div className="message-icon">{getIcon()}</div>
      <div className="message-content">
        {message.content}
        {message.metadata?.videoUrl && (
          <div className="message-attachment">
            <video src={message.metadata.videoUrl} controls />
          </div>
        )}
      </div>

};
```

```tsx
// frontend/src/components/InfiniteCanvas/InfiniteCanvas.tsx

import React, { useState, useRef, useEffect, useCallback } from 'react';
import './InfiniteCanvas.css';

interface Node {
  id: string;
  type: 'script' | 'scene' | 'material' | 'video' | 'intent' | 'plan' | 'operation';
  data: any;
  position: { x: number; y: number };
  size: { width: number; height: number };
  style?: any;
  selected?: boolean;
}

interface Connection {
  id: string;
  sourceNodeId: string;
  targetNodeId: string;
  connectionType: string;
  label?: string;
}

const InfiniteCanvas: React.FC<{ projectId: string }> = ({ projectId }) => {
  const canvasRef = useRef<HTMLDivElement>(null);
  const [nodes, setNodes] = useState<Node[]>([]);
  const [connections, setConnections] = useState<Connection[]>([]);
  const [selectedNodeId, setSelectedNodeId] = useState<string | null>(null);
  const [viewport, setViewport] = useState({ x: 0, y: 0, scale: 1 });
  const [isDragging, setIsDragging] = useState(false);
  const [dragStart, setDragStart] = useState({ x: 0, y: 0 });
  const wsRef = useRef<WebSocket | null>(null);

  // WebSocket 连接
  useEffect(() => {
    wsRef.current = new WebSocket(`ws://${location.host}/ws/canvas/${projectId}`);
    
    wsRef.current.onmessage = (event) => {
      const data = JSON.parse(event.data);
      handleCanvasEvent(data);
    };

    // 加载初始数据
    loadCanvasData();

    return () => {
      wsRef.current?.close();
    };
  }, [projectId]);

  const loadCanvasData = async () => {
    const [nodesRes, connectionsRes] = await Promise.all([
      fetch(`/api/canvas/nodes/${projectId}`),
      fetch(`/api/canvas/connections/${projectId}`)
    ]);
    
    const nodesData = await nodesRes.json();
    const connectionsData = await connectionsRes.json();
    
    setNodes(nodesData);
    setConnections(connectionsData);
  };

  const handleCanvasEvent = (event: any) => {
    switch (event.type) {
      case 'node_created':
        setNodes(prev => [...prev, event.node]);
        break;
        
      case 'node_updated':
        setNodes(prev => prev.map(n => 
          n.id === event.nodeId ? { ...n, data: event.newData } : n
        ));
        break;
        
      case 'node_deleted':
        setNodes(prev => prev.filter(n => n.id !== event.nodeId));
        setConnections(prev => prev.filter(c => 
          c.sourceNodeId !== event.nodeId && c.targetNodeId !== event.nodeId
        ));
        break;
        
      case 'connection_created':
        setConnections(prev => [...prev, event.connection]);
        break;
    }
  };

  // 缩放
  const handleWheel = useCallback((e: WheelEvent) => {
    e.preventDefault();
    const delta = e.deltaY > 0 ? 0.9 : 1.1;
    const newScale = Math.max(0.1, Math.min(3, viewport.scale * delta));
    
    const rect = canvasRef.current!.getBoundingClientRect();
    const mouseX = e.clientX - rect.left;
    const mouseY = e.clientY - rect.top;
    
    setViewport(prev => ({
      x: mouseX - (mouseX - prev.x) * (newScale / prev.scale),
      y: mouseY - (mouseY - prev.y) * (newScale / prev.scale),
      scale: newScale
    }));
  }, [viewport]);

  // 平移
  const handleMouseDown = useCallback((e: React.MouseEvent) => {
    if (e.target === canvasRef.current || (e.target as HTMLElement).classList.contains('canvas-background')) {
      setIsDragging(true);
      setDragStart({ x: e.clientX - viewport.x, y: e.clientY - viewport.y });
      setSelectedNodeId(null);
    }
  }, [viewport]);

  const handleMouseMove = useCallback((e: React.MouseEvent) => {
    if (isDragging) {
      setViewport(prev => ({
        ...prev,
        x: e.clientX - dragStart.x,
        y: e.clientY - dragStart.y
      }));
    }
  }, [isDragging, dragStart]);

  const handleMouseUp = useCallback(() => {
    setIsDragging(false);
  }, []);

  // 节点拖拽
  const handleNodeDrag = useCallback((nodeId: string, newPosition: { x: number; y: number }) => {
    setNodes(prev => prev.map(n => 
      n.id === nodeId ? { ...n, position: newPosition } : n
    ));
    
    // 同步到服务器
    fetch(`/api/canvas/nodes/${nodeId}/position`, {
      method: 'PUT',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify(newPosition)
    });
  }, []);

  return (
    <div 
      className="infinite-canvas"
      ref={canvasRef}
      onWheel={handleWheel}
      onMouseDown={handleMouseDown}
      onMouseMove={handleMouseMove}
      onMouseUp={handleMouseUp}
      onMouseLeave={handleMouseUp}
    >
      {/* 网格背景 */}
      <div 
        className="canvas-background"
        style={{
          backgroundPosition: `${viewport.x}px ${viewport.y}px`,
          backgroundSize: `${20 * viewport.scale}px ${20 * viewport.scale}px`
        }}
      />

      {/* 变换容器 */}
      <div 
        className="canvas-transform"
        style={{
          transform: `translate(${viewport.x}px, ${viewport.y}px) scale(${viewport.scale})`
        }}
      >
        {/* 连接线 */}
        <svg className="connections-layer">
          {connections.map(conn => (
            <ConnectionLine
              key={conn.id}
              connection={conn}
              nodes={nodes}
            />
          ))}
        </svg>

        {/* 节点 */}
        {nodes.map(node => (
          <CanvasNode
            key={node.id}
            node={node}
            isSelected={selectedNodeId === node.id}
            onSelect={() => setSelectedNodeId(node.id)}
            onDrag={(pos) => handleNodeDrag(node.id, pos)}
          />
        ))}
      </div>

      {/* 工具栏 */}
      <div className="canvas-toolbar">
        <Button onClick={() => setViewport({ x: 0, y: 0, scale: 1 })}>
          重置视图
        </Button>
        <Slider
          min={0.1}
          max={3}
          step={0.1}
          value={viewport.scale}
          onChange={(scale) => setViewport(prev => ({ ...prev, scale }))}
          style={{ width: 120 }}
        />
        <span>{Math.round(viewport.scale * 100)}%</span>
      </div>

      {/* 迷你地图 */}
      <MiniMap nodes={nodes} viewport={viewport} />
    </div>
  );
};

// 节点组件
const CanvasNode: React.FC<{
  node: Node;
  isSelected: boolean;
  onSelect: () => void;
  onDrag: (pos: { x: number; y: number }) => void;
}> = ({ node, isSelected, onSelect, onDrag }) => {
  const nodeRef = useRef<HTMLDivElement>(null);
  const [isDragging, setIsDragging] = useState(false);
  const [dragOffset, setDragOffset] = useState({ x: 0, y: 0 });

  const handleMouseDown = (e: React.MouseEvent) => {
    e.stopPropagation();
    setIsDragging(true);
    setDragOffset({
      x: e.clientX - node.position.x,
      y: e.clientY - node.position.y
    });
    onSelect();
  };

  const handleMouseMove = (e: MouseEvent) => {
    if (isDragging) {
      onDrag({
        x: e.clientX - dragOffset.x,
        y: e.clientY - dragOffset.y
      });
    }
  };

  const handleMouseUp = () => {
    setIsDragging(false);
  };

  useEffect(() => {
    if (isDragging) {
      window.addEventListener('mousemove', handleMouseMove);
      window.addEventListener('mouseup', handleMouseUp);
      return () => {
        window.removeEventListener('mousemove', handleMouseMove);
        window.removeEventListener('mouseup', handleMouseUp);
      };
    }
  }, [isDragging, dragOffset]);

  const getNodeComponent = () => {
    switch (node.type) {
      case 'script': return <ScriptNode data={node.data} />;
      case 'scene': return <SceneNode data={node.data} />;
      case 'material': return <MaterialNode data={node.data} />;
      case 'video': return <VideoNode data={node.data} />;
      case 'intent': return <IntentNode data={node.data} />;
      case 'plan': return <PlanNode data={node.data} />;
      case 'operation': return <OperationNode data={node.data} />;
      default: return <div>Unknown</div>;
    }
  };

  return (
    <div
      ref={nodeRef}
      className={`canvas-node node-${node.type} ${isSelected ? 'selected' : ''}`}
      style={{
        left: node.position.x,
        top: node.position.y,
        width: node.size?.width || 200,
        height: node.size?.height || 150,
        borderColor: node.style?.borderColor,
        backgroundColor: node.style?.backgroundColor,
        zIndex: isSelected ? 100 : 1
      }}
      onMouseDown={handleMouseDown}
      onDoubleClick={() => {/* 打开编辑器 */}}
    >
      {getNodeComponent()}
    </div>
  );
};

const ConnectionLine: React.FC<{ connection: Connection; nodes: Node[] }> = ({ connection, nodes }) => {

  const x1 = sourceNode.position.x + (sourceNode.size?.width || 200) / 2;
  const y1 = sourceNode.position.y + (sourceNode.size?.height || 150);
  const x2 = targetNode.position.x + (targetNode.size?.width || 200) / 2;
  const y2 = targetNode.position.y;

  const getLineStyle = () => {
    switch (connection.connectionType) {
      case 'timeline': return { stroke: '#8c8c8c', strokeDasharray: '0' };
      case 'reference': return { stroke: '#1890ff', strokeDasharray: '5,5' };
      case 'dependency': return { stroke: '#fa8c16', strokeDasharray: '0' };
      case 'operation': return { stroke: '#722ed1', strokeDasharray: '2,2' };
      default: return { stroke: '#d9d9d9', strokeDasharray: '5,5' };
    }
  };

  const style = getLineStyle();

  return (
    <g>
      <line
        x1={x1} y1={y1}
        x2={x2} y2={y2}
        stroke={style.stroke}
        strokeWidth={2}
        strokeDasharray={style.strokeDasharray}
        markerEnd="url(#arrowhead)"
      />
      {connection.label && (
        <text
          x={(x1 + x2) / 2}
          y={(y1 + y2) / 2}
          textAnchor="middle"
          fontSize={10}
          fill="#595959"
        >

      )}

        />
      </div>

};
```

## 十、优势总结

### 10.1 技术优势

| 特性 | 描述 |
|------|------|
| **可视化思维链** | Agent 的意图→规划→执行全程可视化 |
| **双向实时同步** | 数据库更新即时反映到画布 |
| **异步确认机制** | 危险操作需用户确认，保障安全 |
| **自由布局** | 无限画布支持任意节点拖拽和组织 |
| **自然语言交互** | 用户无需学习，对话即操作 |

### 10.2 用户体验优势

- **降低学习成本**：无需理解复杂流程，开口即用
- **提升透明度**：Agent 干了什么，一目了然
- **增强可控性**：关键操作需确认，防止误操作
- **提高效率**：一个指令完成多步操作

### 10.3 架构优势

- **模块化设计**：意图识别、任务规划、工具调度解耦
- **可扩展性**：易于添加新的意图类型和工具
- **可观测性**：完整操作历史，便于调试和审计
- **前后端分离**：清晰的 API 接口，松耦合
