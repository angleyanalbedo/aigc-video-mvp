# Settings 页面使用指南

## 📍 访问位置

Settings 页面位于侧边栏底部 ⚙️ 图标，或者直接访问：
```
http://localhost:5173/settings
```

## 🎯 功能模块

Settings 页面包含 **4 个主要模块**，通过 Tab 标签页切换：

### 1️⃣ 系统配置 (System Configuration)

配置系统级参数：

**API 配置**
- 🔑 **API 密钥** - 火山方舟 API 密钥，用于调用大模型
- 🌐 **API 端点** - API 服务器地址（默认已配置）

**视频生成默认值**
- 📐 **画幅比例** - 9:16 (竖版), 16:9 (横版), 1:1 (方形), 4:3
- ⏱️ **默认时长** - 5秒 / 10秒 / 15秒 / 20秒 / 30秒
- 🎬 **分辨率** - 480p / 720p

**自动保存设置**
- 🔄 **启用自动保存** - 开关控制
- ⏰ **保存间隔** - 15秒 / 30秒 / 60秒 / 120秒

### 2️⃣ 创作默认值 (Creative Defaults)

设置视频生成时的创作因子默认值，这些值将在创建新视频时自动填充：

**开场与叙事**
- 🎬 **开场方式** - 直接展示 / 痛点提问 / 悬念引入 / 场景代入 / 故事引入
- 🎙️ **旁白风格** - 活泼热情 / 知性优雅 / 专业权威 / 亲切自然

**视觉风格**
- 🎨 **画面色调** - 暖色调 / 冷色调 / 高饱和 / 低饱和 / 中性
- 📝 **字幕样式** - 大字醒目 / 底部标准 / 动感字幕 / 无字幕

**音频与节奏**
- 🎵 **BGM 风格** - 节奏感强 / 轻快 / 温馨 / 紧张 / 科技 / 无BGM
- ⏱️ **视频时长** - 同上
- 🎞️ **分镜数量** - 1个 / 2个 / 3个 / 5个 / 8个

### 3️⃣ 数据管理 (Data Management)

管理本地数据和数据库：

**本地数据**
- 📊 系统配置状态
- 🎨 创作默认值状态
- 💾 项目草稿保存状态

**操作按钮**
- 🔄 **初始化示例数据** - 重新生成示例视频数据
- 🗑️ **清除浏览器缓存** - 清除所有 localStorage 和 sessionStorage
- ⚙️ **重置所有设置** - 恢复系统默认配置

**数据库**
- 📁 数据库位置：`server/data/app.db`
- 📈 查看数据库统计

## 💾 数据存储

### localStorage
- `systemConfig` - 系统配置
- `creativeDefaults` - 创作默认值

### sessionStorage
- 临时会话数据

## 🔧 常见操作

### 保存配置
1. 切换到对应 Tab
2. 修改配置项
3. 点击 "保存配置" 或 "保存默认值" 按钮
4. 显示成功提示 ✓

### 重置配置
1. 点击 "重置" 按钮
2. 确认对话框
3. 配置恢复为默认值

### 清除缓存
1. 点击 "清除浏览器缓存"
2. 确认对话框
3. 缓存清除完成，需要刷新页面

## ⚠️ 注意事项

1. **配置保存** - 所有配置保存在浏览器 localStorage，不会同步到服务器
2. **API 密钥** - 敏感信息，请勿泄露给他人
3. **数据安全** - 清除缓存会删除所有本地保存的配置和草稿
4. **重置操作** - 重置后需要重新配置系统参数

## 🎯 最佳实践

### 首次使用
1. 访问 Settings 页面
2. 配置 API 密钥和端点
3. 设置创作默认值
4. 点击 "初始化示例数据" 生成演示数据

### 日常使用
1. 保持默认配置不变
2. 在创建视频时根据需要调整
3. 定期查看系统状态

### 数据管理
1. 定期备份重要数据
2. 不需要的数据及时清理
3. 重置前确认操作

## 📞 支持

2. 检查浏览器控制台错误信息
3. 清除缓存后重试
