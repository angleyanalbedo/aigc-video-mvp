## 四、具体实现方案

### 4.1 生成过程Trace（推荐优先实现）

**目的**：记录整个生成过程，便于调试和展示

```javascript
// server/services/traceService.js
class TraceService {
  constructor() {
    this.traces = new Map();
  }

  startTrace(taskId) {
    this.traces.set(taskId, {
      id: taskId,
      startTime: Date.now(),
      steps: []
    });
  }

  addStep(taskId, step, data) {
    const trace = this.traces.get(taskId);
    if (trace) {
      trace.steps.push({
        step,
        timestamp: Date.now(),
        data
      });
    }
  }

  getTrace(taskId) {
    return this.traces.get(taskId);
  }
}

// 使用示例
traceService.addStep(taskId, 'script_generated', { scenes: 4 });
traceService.addStep(taskId, 'video_scene_1', { url: '...' });
```

**API 端点**：
```
GET /api/tasks/:id/trace
```

### 4.2 失败重试机制

**目的**：视频/TTS生成失败时自动重试

```javascript
// server/utils/retry.js
async function withRetry(fn, options = {}) {
  const { maxRetries = 3, delay = 1000, backoff = 2 } = options;
  
  for (let i = 0; i < maxRetries; i++) {
    try {
      return await fn();
    } catch (error) {
      if (i === maxRetries - 1) throw error;
      
      console.log(`重试 ${i + 1}/${maxRetries}，等待 ${delay}ms`);
      await sleep(delay);
      delay *= backoff; // 指数退避
    }
  }
}

// 使用示例
const videoUrl = await withRetry(() => generateVideo(scene), { maxRetries: 3 });
```

### 4.3 SSE 实时推送

**目的**：替代轮询，提升用户体验

```javascript
// server/index.js
app.get('/api/tasks/:id/stream', (req, res) => {
  res.setHeader('Content-Type', 'text/event-stream');
  res.setHeader('Cache-Control', 'no-cache');
  res.setHeader('Connection', 'keep-alive');

  const taskId = req.params.id;
  
  const interval = setInterval(() => {
    const task = taskStore.get(taskId);
    res.write(`data: ${JSON.stringify(task)}\n\n`);
    
    if (task.status === 'completed' || task.status === 'failed') {
      clearInterval(interval);
      res.end();
    }
  }, 1000);

  req.on('close', () => clearInterval(interval));
});
```

**前端使用**：
```typescript
const eventSource = new EventSource(`/api/tasks/${taskId}/stream`);
eventSource.onmessage = (event) => {
  const data = JSON.parse(event.data);
  setProgress(data.progress);
  if (data.status === 'completed') {
    setVideoUrl(data.videoUrl);
    eventSource.close();
  }
};
```
