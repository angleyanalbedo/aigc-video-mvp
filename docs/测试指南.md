# 智能无限画布工作台 - 测试指南

## 🚀 快速测试步骤

### 1. 启动服务

#### 后端
```bash
cd server
node index.js
```
确保显示：`🚀 服务器运行在 http://localhost:3001`

#### 前端
```bash
cd frontend
npm run dev
```
确保显示类似：`Local: http://localhost:5173/`

---

## 📝 测试流程

### 方式一：使用前端界面

#### 步骤 1：创建项目
1. 打开浏览器访问 http://localhost:5173
2. 点击 **新建项目**
3. 输入项目名称（如：测试项目）

#### 步骤 2：进入 Copilot 工作台
1. 在项目卡片上点击 🤖 **Copilot AI 助手** 按钮
2. 进入智能无限画布工作台页面

#### 步骤 3：测试对话功能
1. 在左侧聊天栏输入：`生成视频`
2. 应该会看到：
   - ✅ 创建了 Intent 节点
   - ✅ 创建了 Plan 节点
   - ✅ 显示确认卡片

#### 步骤 4：确认执行
1. 点击 **确认执行** 按钮
2. 应该会看到：
   - ✅ 创建了 Operation 节点
   - ✅ 节点显示在右侧画布上

#### 步骤 5：查看画布
1. 在右侧画布上应该能看到节点
2. 缩放：鼠标滚轮
3. 拖拽节点：直接拖拽
4. 重置视图：点击左下角的重置按钮

---

### 方式二：使用 API 测试

#### 步骤 1：创建项目（浏览器控制台）
```javascript
// 在浏览器控制台执行
const res = await fetch('/api/projects', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ name: '测试项目' })
});
const project = await res.json();
console.log('项目ID:', project.data.id);
```

#### 步骤 2：创建会话
```javascript
const session = await fetch('/api/copilot/chat/sessions', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ projectId: '项目ID', title: '测试' })
});
const s = await session.json();
console.log('会话ID:', s.sessionId);
```

#### 步骤 3：发送消息
```javascript
const chat = await fetch('/api/copilot/chat', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    message: '生成视频',
    projectId: '项目ID',
    sessionId: '会话ID'
  })
});
const result = await chat.json();
console.log('结果:', result);
```

#### 步骤 4：查看节点
```javascript
const nodes = await fetch('/api/copilot/canvas/nodes/项目ID');
const n = await nodes.json();
console.log('节点:', n.nodes);
```

## 🐛 常见问题

### ❌ 节点不显示
**原因**：页面没有刷新  
**解决**：发送消息后，**刷新页面**

### ❌ 点击按钮没反应
**原因**：还在加载中  
**解决**：等待 500ms 后重试

### ❌ API 返回 404
**原因**：后端未启动  
**解决**：确保后端运行在 3001 端口

### ❌ 缩放无效
**原因**：焦点在输入框  
**解决**：点击画布背景后再缩放

## 🎯 推荐测试序列

1. **基础测试**：发送"现在状态如何"
   - 应该：直接执行，不需要确认
   - 应该：在画布上创建 3 个节点

2. **确认流程**：发送"生成视频"
   - 应该：显示确认卡片
   - 应该：在画布上创建 Intent + Plan 节点

3. **执行流程**：点击"确认执行"
   - 应该：创建 Operation 节点
   - 应该：节点出现在画布上

4. **交互测试**：
   - 缩放画布
   - 拖拽节点
   - 查看迷你地图

## 📞 获取帮助

如果测试失败：
1. 检查后端控制台是否有错误
2. 检查浏览器控制台（F12）
3. 查看 Network 标签页
4. 刷新页面重试
