## 项目概述

**智能无限画布工作台**是一个 AI 驱动的视频创作平台，通过自然语言对话即可完成视频生成、编辑和编辑等操作，配合无限画布可视化 Agent 完整的思考链。

### 系统架构

```
┌─────────────────────────────────────────────────────────┐
│                    前端 (React + Ant Design)              │
├─────────────────────────────────────────────────────────┤
│  ┌─────────────────┐  ┌──────────────────────────────┐ │
│  │  Copilot Chat  │  │      Infinite Canvas        │ │
│  │   (对话界面)   │  │       (无限画布)          │ │
│  └─────────────────┘  └──────────────────────────────┘ │
└─────────────────────────────────────────────────────────┘
                          ↓
┌─────────────────────────────────────────────────────────┐
│           MasterAgent (后端 (Node.js + Express)             │
├─────────────────────────────────────────────────────────┤
│  ┌──────────────────────────────────────────────────────┐  │
│  │   Intent Parser → Task Planner → Tool Executor     │  │
│  └──────────────────────────────────────────────────────┘  │
└─────────────────────────────────────────────────────────┘
                          ↓
┌─────────────────────────────────────────────────────────┐
│            SQLite Database + Canvas Nodes/Connections/Operations     │
└─────────────────────────────────────────────────────────┘
```

## 功能特性

### ✨ 核心功能

| 功能 | 状态 | 说明 |
|------|------|------|
| 自然语言对话 | ✅ | 用中文下达创作意图 |
| 意图识别 | ✅ | 自动理解指令意图和执行计划 |
| 无限画布 | ✅ | 无限网格背景、节点拖拽、缩放平移 |
| 实时同步 | ✅ | WebSocket 实时更新 |
| 异步确认 | ✅ | 关键操作需要确认 |
| 操作可视化 | ✅ | Intent → Plan → Operation 节点可视化 |

### 🎯 节点类型

- **Intent 节点**：用户意图识别结果
- **Plan 节点**：AI 生成的执行计划
- **Operation 节点**：单个执行步骤
- **Script 节点**：项目剧本
- **Scene 节点**：分镜节点
- **Material 节点**：素材节点
- **Video 节点**：生成的视频

## 快速开始

### 1. 启动后端

#### 安装依赖

```bash
cd server
npm install
```

#### 启动服务器

```bash
cd server
node index.js
```

服务器将在 http://localhost:3001 启动

### 2. 启动前端

#### 安装依赖

```bash
cd frontend
npm install
```

#### 启动开发服务器

```bash
npm run dev
```

前端将在 http://localhost:5173 或类似的端口启动

### 3. 访问应用

1. 在前端页面访问项目列表页，创建一个新项目
2. 在项目卡片上点击 🤖 **Copilot AI 助手** 按钮
3. 进入智能无限画布工作台！

## 核心功能使用

### 1. 对话界面

#### 开始对话

在左侧聊天栏输入你的创作：
- **示例 1：**`生成一个破壁机产品的视频
- **示例 2：**`把第 2 个分镜改成特写
- **示例 3：**`现在的状态怎么样

#### 快捷操作确认

生成类或复杂操作时，会显示确认卡片：
- 点击 **确认执行**：开始执行
- 点击 **取消**：放弃操作

#### 建议按钮

首次进入时可以看到快捷建议：
- 帮我生成产品视频
- 查询状态
- 修改分镜

#### 历史记录

- 左侧聊天保存历史记录
- 系统提示显示操作过程

---

### 2. 无限画布

#### 基础操作

| 操作 | 方法 |
|------|------|
| 缩放 | 鼠标滚轮 |

| 节点移动 | 直接拖拽节点 |
| 节点选中 | 点击节点 |

#### 工具栏

- 左下角的工具栏：
- 🔄 **重置视图**
- 🔍 **缩放**
- 📏 **缩放滑块**
- 🔍 **放大**

#### 迷你地图

- 右下角显示整个画布缩略图
- 蓝色框显示当前视口位置

## API 接口文档

### Copilot API

| 端点 | 方法 | 说明 |
|------|------|
| /api/copilot/chat | POST | 发送用户消息 |
| /api/copilot/execute | POST | 执行确认的计划 |
| /api/copilot/cancel | POST | 取消计划 |
| /api/copilot/canvas/nodes/:projectId | GET | 获取节点列表 |
| /api/copilot/canvas/connections/:projectId | GET | 获取节点连接 |
| /api/copilot/canvas/nodes/:nodeId/position | PUT | 更新节点位置 |
| /api/copilot/chat/sessions/:sessionId/messages | GET | 获取会话历史 |
| /api/copilot/chat/sessions | POST | 创建会话 |

### WebSocket

- ws://localhost:3001/ws/canvas/:projectId

连接后会实时接收画布更新
