## 一、项目概述

### 课题背景
- TikTok Shop 国际电商快速发展
- 短视频是商家获取流量和提升成交的关键载体
- 目标：打造面向商家的 AIGC 带货视频生成系统

### 核心目标
端到端自动生成带货视频，支持：
- 素材导入与管理
- Prompt 调整
- 分镜干预
- 素材混剪

## 三、核心功能模块

### 核心流程
```
素材库建设 → 剧本生成 → 视频创作 → 数据回流反哺（可选）
```

### 3.1 素材模块

**目标**：给 AIGC 视频生成提供有效素材（图像、视频片段）

| 功能 | 说明 |
|------|------|
| 素材入库 | 支持上传商品主图、商品视频、参考素材 |
| 多颗粒度结构化 | 切片处理 + 多模态理解，提取商品/视频/切片维度特征 |
| 素材检索 | 基于关键词、标签、向量相似度的多颗粒度检索 |

### 3.2 剧本模块

**核心流程**：找参考 → 提炼方法论 → 生产剧本

| 功能 | 说明 |
|------|------|
| 优质视频库 | 按类目/关键词检索站外爆款视频，结构化拆解报告 |
| 方法论提炼 | 聚类爆款视频，归纳为创作模板（策略 + 因子） |
| 剧本生成 | 输入商品信息，生成完整剧本（叙事框架、视觉风格、分镜脚本） |
| 生产模式 | 爆款仿写 / 灵感模板 / 剧本自动化（进阶） |
| 剧本干预 | Prompt 微调、分镜增删、台词改写、因子局部替换 |

**剧本输出内容**：
- 叙事框架
- 视觉风格
- 分镜脚本（画面描述、镜头运动、BGM、字幕）
- 约束清单

### 3.3 创作模块

**视频时长**：不超过 15s

| 功能 | 说明 |
|------|------|
| 一键成片 | 输入商品链接或主图，端到端产出完整带货视频 |
| 智能剪辑 | 画面拼接、转场、字幕、配音（多语种）、配乐合成 |
| 分镜级干预 | 单个分镜重生成、替换素材切片、调整时长、修改台词 |
| 预览与导出 | 多分辨率（480p/720p）、多画幅（9:16/16:9/1:1/4:3） |

## 八、FAQ 要点

| 问题 | 回答 |
|------|------|
| 技术栈必须严格遵守吗？ | 核心技术栈（React、Node.js、TypeScript、火山引擎）建议保持一致；UI 组件库、图表库、Agent 框架可自由选择 |
| 视频生成必须端到端跑通吗？ | 基础要求是端到端可运行；视频效果不是评审重点，更看重工程链路完整性和创新思路 |
| 数据从哪里来？ | 可使用 mock 数据或自建模拟服务，重点在数据结构设计、归因逻辑与可视化 |
| 素材合规与版权如何处理？ | 使用公开可商用素材或自有素材，重点是设计"合规审核 + 真实性校验"工作流 |
| 可以借用 AI 编程吗？ | 完全可以，推荐使用 Trae 等 AI 编程工具 |
| 可以使用第三方库吗？ | 完全可以，开源社区的任何工具和方案都可以使用 |
