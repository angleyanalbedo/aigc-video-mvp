# 开源项目参考汇总 - AIGC 视频生成

> 电商场景 AIGC 带货视频生成系统 - 参考开源项目整理

---

## 一、完整工作流项目（推荐参考）

### 1. Toonflow ⭐ 8.1k
- **GitHub**: https://github.com/HBAI-Ltd/Toonflow-app
- **定位**: 开源一站式 AI 短剧创作工具
- **核心功能**:
  - 小说/剧本 → 动画短剧自动转换
  - AI 编剧、智能分镜
  - 角色与视频生成
  - 跨平台桌面端应用（Electron + Vue3）
- **技术栈**: Electron + Node.js + Vue3 + TypeScript
- **可借鉴点**:
  - 小说→剧本→分镜→视频的全流程架构
  - 多 Agent 协同架构
  - 本地数据库控制角色一致性
  - 分镜时间轴编辑器 UI

---

### 2. ArcReel ⭐ 2.3k
- **GitHub**: https://github.com/ArcReel/ArcReel
- **定位**: AI Agent 驱动的开源视频生成工作台
- **核心功能**:
  - 小说 → 角色/场景/道具设计 → 剧本 → 分镜图 → 视频
  - 跨镜头角色与场景一致性保持
- **支持模型**: Nano Banana 2、Veo 3.1、Grok、Seedance、OpenAI
- **技术栈**: Python + Docker
- **可借鉴点**:
  - AI Agent 编排架构
  - 多模型集成方案
  - 角色一致性保持机制

---

### 3. Story-Flicks ⭐ 2.3k
- **GitHub**: https://github.com/alecm20/story-flicks
- **定位**: 一键生成高清故事短视频
- **核心功能**:
  - 使用 AI 大模型自动生成故事短视频
  - 端到端视频生成流程
- **技术栈**: Python + OpenAI + MoviePy
- **可借鉴点**:
  - 简单的端到端视频生成流程
  - Python 后端架构
  - Prompt 工程方案

---

### 4. MovieStudio
- **GitHub**: https://github.com/alessoh/MovieStudio
- **定位**: AI 电影制作工作室（多 Agent）
- **核心功能**:
  - Screenplay Writer Agent (剧本编写)
  - Visual Development Artist Agent (视觉开发)
  - Character Designer Agent (角色设计)
  - Storyboard Artist Agent (分镜设计)
  - Video Generation Agent (视频生成)
  - Sound Designer Agent (音效设计)
- **技术栈**: Python
- **可借鉴点**:
  - 多 Agent 协作架构设计
  - 影视制作流程抽象

## 二、基础模型框架

### 5. Wan 2.1（阿里巴巴）
- **GitHub**: https://github.com/Wan-Video/Wan2.1
- **定位**: 阿里开源的高性能视频生成模型
- **性能**: VBench 评测 86.2 分，排名第一
- **功能**:
  - 文生视频 (T2V)
  - 图生视频 (I2V)
  - 视频编辑
  - 动态文字特效
- **硬件要求**: 1.3B 版本仅需 8.19GB 显存
- **分辨率**: 支持 480P 和 720P

---

### 6. HunyuanVideo（腾讯）
- **GitHub**: https://github.com/Tencent/HunyuanVideo
- **定位**: 腾讯开源视频生成大模型
- **特点**: 130 亿参数，目前参数量最大的开源视频模型之一
- **技术创新**:
  - 3D-VAE 编解码技术
  - 全注意力机制
  - 多模态文本编码
- **功能**: 文生视频、图生视频、视频编辑、动态文字特效

---

### 7. Open-Sora（HPC-AI Tech）
- **GitHub**: https://github.com/hpcaitech/Open-Sora
- **定位**: 最早复现 Sora 的开源项目之一
- **性能**: 可生成最长 16 秒、720p 分辨率视频
- **架构**: Spatial-Temporal Diffusion Transformer (STDiT)
- **功能**: 文生图、文生视频、图生视频

---

### 8. VGen（阿里达摩院）
- **GitHub**: https://github.com/ali-vilab/VGen
- **定位**: 完整的视频生成生态系统
- **基础**: 基于扩散模型 (Diffusion Models)
- **Stars**: 3.2k

---
