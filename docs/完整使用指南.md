## 系统概述

基于火山方舟 API 的 AI 带货视频生成系统，包含**赛题核心模块**：

| 功能模块 | 说明 |
|---------|------|
| 📚 **优质视频库** | 爆款视频素材库 + AI 结构化分析 |
| ⚡ **灵感模板库** | 方法论提炼 + 策略+因子创作模板 |
| 🎨 **视频创作** | 一键生成 + 手动精修 + 分镜编辑 |
| 🤖 **Copilot AI** | 智能无限画布 + 自然语言创作 |
| 📊 **A/B 测试** | 策略效果对比 |
| 📈 **多因子归因** | 视频效果分析 |

## 核心功能详解

---

### 📚 优质视频库

#### 功能说明
保存爆款视频的**结构化分析结果**（不含原视频），用于仿写参考。

#### 使用流程

1. **添加入库**
   - 点击「添加入库」按钮
   - 填写视频信息：标题、平台、类目、来源声明
   - 选填：视频链接、封面图、标签

2. **AI 分析**
   - 点击「AI分析」按钮
   - 系统自动生成 5 维度分析：
     - **Hook 手法**：开场3秒如何抓眼球
     - **卖点呈现**：产品亮点如何展示
     - **分镜分析**：镜头节奏和运用
     - **风格分析**：视觉风格、色调、BGM
     - **结构分析**：叙事框架

3. **筛选检索**
   - 按类目筛选：美妆、食品、家居、数码、运动
   - 按关键词搜索：标题、标签、卖点
   - 按平台筛选：抖音、小红书、TikTok、Instagram

#### 数据字段
```javascript
{
  title: "美妆口红试色短视频",
  platform: "TikTok",
  category: "美妆",
  tags: ["口红", "试色", "种草"],
  hookTechnique: "第一人称视角直接展示上嘴效果",
  sellingPoints: "持久不脱色、滋润不拔干",
  shotAnalysis: "特写→中景→特写，3秒一个节奏点",
  styleAnalysis: "暖色调、近距离拍摄、自然光线",
  structureAnalysis: "开场Hook→色号展示→效果对比→CTA",
  sourceDeclaration: "公开平台热门视频结构化分析"
}
```

### ⚡ 灵感模板库

#### 功能说明
从多个爆款视频中**聚类提炼方法论**，生成「策略+因子」创作模板。

#### 使用流程

1. **从爆款视频聚类提炼**
   - 点击「从爆款视频聚类提炼」
   - 选择至少 2 个已分析的视频
   - 系统自动归纳共同套路
   - 生成结构化模板

2. **手动创建模板**
   - 点击「创建模板」
   - 填写策略和因子：
     - **策略**：抽象创作方法，如"第一人称 BGM 氛围沉浸"
     - **因子**：具体创作手段（6个维度）：
       - 开场因子
       - 退场因子
       - 画面因子
       - 旁白因子
       - BGM因子
       - 色调因子

#### 模板结构
```javascript
{
  name: "痛点对比反转模板",
  category: "家居",
  strategy: "展示杂乱场景引发共鸣，3秒后展示收纳后效果形成强烈对比",
  factors: {
    opening: "痛点展示+夸张表情",
    closing: "黑屏品牌名+限时优惠",
    visual: "远景→特写→中景→远景，对比镜头为主",
    voiceover: "急促有力+引导下单",
    bgm: "节奏感强的电子乐",
    color_tone: "冷色调高级感"
  },
  constraintRules: "适用于家居收纳产品，视频时长15-20秒"
}
```

### 🎨 视频创作

#### 模式 1: 一键生成
1. 进入「一键生成」页面
2. 输入产品信息：名称、卖点、目标人群
3. 选择风格和分辨率
4. 点击「一键生成」

#### 模式 2: 模板生成（推荐）
1. 从「灵感模板库」选择模板
2. 点击「作为参考创作」
3. 自动填充策略和因子
4. 微调后生成视频

#### 模式 3: 手动工作流
1. **生成剧本**
   - 输入产品信息
   - 选择模板/参考视频
   - AI 生成剧本和分镜

2. **剧本干预**
   - 修改旁白文案
   - 调整分镜描述
   - 增减分镜数量

3. **编辑分镜**
   - 修改画面描述
   - 调整时长
   - 上传素材图片

4. **生成视频**
   - 点击「生成视频」
   - 等待约 5-10 分钟

### 🤖 Copilot AI 助手

#### 功能说明
自然语言对话 + 无限画布可视化 Agent 思考链。

#### 使用示例

| 指令 | 效果 |
|------|------|
| "生成一个防晒霜的视频" | 自动创建项目 → 生成剧本 → 生成分镜 → 生成视频 |
| "把第2个分镜改成特写" | 找到第2个分镜 → 修改描述 → 更新剧本 |
| "现在的状态怎么样" | 展示当前项目状态、分镜数、素材数 |
| "用痛点对比模板创作" | 加载模板 → 应用策略 → 生成内容 |

---

### 📊 A/B 测试

#### 功能说明
对比不同创作策略的效果差异。

#### 使用流程
1. 创建实验，设置对照组和实验组
2. 为两组设置不同策略（开场方式、BGM风格等）
3. 发布视频后查看效果对比
4. 分析完播率、转化率差异

---

### 📈 多因子归因

#### 功能说明
分析哪个创作因子对视频效果影响最大。

#### 分析维度
| 因子 | 说明 |
|------|------|
| 开场方式 | 悬念引入、痛点提问、故事引入等 |
| BGM风格 | 节奏感强、轻快、温馨等 |
| 旁白风格 | 活泼热情、优雅知性、专业等 |
| 色调 | 暖色调、冷色调、高饱和等 |
| 时长 | 15秒、20秒、30秒 |
| 分镜数 | 3镜、5镜、7镜 |

## 完整工作流

### 🎯 爆款仿写工作流（推荐）

```
1️⃣ 优质视频库
    ↓ 搜索/筛选
2️⃣ 找到合适的爆款视频
    ↓ 查看分析
3️⃣ 理解 Hook 手法、卖点呈现、分镜结构
    ↓ （可选）
4️⃣ 灵感模板库
    ↓ 聚类提炼
5️⃣ 生成创作模板
    ↓ 使用模板
6️⃣ 视频创作
    ↓ 生成视频
7️⃣ 完成
```

## API 文档

### 赛题核心模块 API

| 模块 | 端点 | 方法 | 说明 |
|------|------|------|------|
| **视频库** | `/api/video-library` | GET | 获取视频列表 |
| | `/api/video-library/:id` | GET | 获取单个视频 |
| | `/api/video-library` | POST | 新增视频 |
| | `/api/video-library/:id/analyze` | POST | AI分析视频 |
| **模板库** | `/api/templates` | GET | 获取模板列表 |
| | `/api/templates` | POST | 创建模板 |
| | `/api/templates/extract` | POST | 从视频聚类提炼 |
| | `/api/templates/:id/generate-script` | POST | 用模板生成剧本 |
| **一键生成** | `/api/one-click/generate` | POST | 一键生成视频 |
| | `/api/one-click/task/:taskId` | GET | 查询任务状态 |
| **素材分析** | `/api/material-analysis/:id/analyze` | POST | 分析素材 |
| | `/api/material-analysis/batch-analyze` | POST | 批量分析 |
| **剧本干预** | `/api/scripts/scenes/:projectId` | GET | 获取分镜 |
| | `/api/scripts/scenes/:sceneId/modify` | POST | 修改分镜 |
| | `/api/scripts/:projectId/refine` | POST | 精修剧本 |

## 常见问题

### Q: 视频生成失败怎么办？
A: 检查 API 密钥是否正确、API 额度是否充足，查看后端日志。

### Q: 如何重置数据库？
A: 删除 `server/data/app.db`，重启后端会自动重建。

### Q: 素材上传失败？
A: 检查 `server/uploads/` 目录权限，确保文件大小不超过限制。
